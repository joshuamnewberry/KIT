# Retrospective

## What Went Well

- Successfully developed a Kotlin Multiplatform application.
- Implemented shared business logic between Android and iOS.
- Established a functional notification system.
- Used Jira and GitHub effectively throughout development.

## What Could Be Improved

- Reduce merge conflicts through smaller, more frequent commits.
- Improve project planning for advanced features.

## Lessons Learned

The team gained experience with Kotlin Multiplatform, mobile development, collaborative software engineering, version control, and agile project management practices.
